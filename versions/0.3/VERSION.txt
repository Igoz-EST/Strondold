Версия 0.3 — снимок рабочей сборки (после исправления строгой типизации и ввода).

Содержит: старт в режиме командира (deferred), башни за монеты, UI построек внизу,
ввод через _unhandled_input (башня/камера/Esc) и _physics_process (движение/прыжок/E),
GameState (commander / tower placement), явный тип Variant для ground hit,
lazy-инициализация UI построек в main.gd.

В папке также icon.svg — проект можно открыть в Godot, указав каталог versions/0.3 как корень.
